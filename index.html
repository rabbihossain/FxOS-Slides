WebAPIs & Web Activities?

What the heck is this?

WebAPIs are a set of APIs to connect and interact with some of your device features.

WebAPI

WebAPIs are something like connecting your App to your device...

  • Accelerometer
  • Oreientation Sensor
  • Lignt Sensor
  • GPS
  • & Bla bla bla

Web Activity

Web Activities are something like having your App...

  • Pick an image
  • Share a Music
  • Send SMS
  • Dial a Number
  • & Bla bla bla

Security?

There are three types of App types regarding Firefox OS security model

  1. Plain - Can access only Safe APIs
  2. Privileged - Can access some sensitive APIs with Users Permissions
  3. Certified - Can aceess everything

getUserMedia API

I need permission?

      "permissions": {
                "audio-capture": {
                "description": "Required to capture audio via getUserMedia"
              },
            "video-capture": {
                "description": "Required to capture audio via getUserMedia"
              }
        }

getUserMedia API

       navigator.getUserMedia = navigator.mozGetUserMedia;
       navigator.getUserMedia(
       {
          video:true,
          audio:false
       },
       function(stream) {
          v.src = window.URL.createObjectURL(stream);
          v.play();
       },
       function(error) {
          alert('Something went wrong. (error code ' + error.code + ')');
       }
       );
